CodeLite - open-source, supports C and C++. Works on all major
platforms
NetBeans - supports C, C++, Fortran, and Assembler. Will run on
Windows, Linux and other platforms. It has a drag-and-drop
interface
Komodo - supports Python, PHP, Ruby, Perl, javaScript. Provides
advanced editing capabilities
Cloud 9 - the editor supports over 100 languages. The run menu 
can run jS, Python, Ruby, and PHP. Allows writing, compiling 
and debugging from the cloud, enabling remote team-work.
Eclipse - supports C, C++, Java, Ruby, Perl and more. It is free.
